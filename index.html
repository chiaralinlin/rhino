Scripting

Python in
Grasshopper

Extend Grasshopper's capabilities with Python scripting. Access RhinoCommon geometry types, write custom algorithms, and create bespoke components — all within a familiar programming environment.

  • → Full RhinoCommon Access — All geometry types, transformations, and analysis functions available directly.
  • → Multiple Inputs/Outputs — Define custom I/O types: point, curve, surface, number, boolean, and more.
  • → Python 3 in Rhino 8 — Full Python 3 support with numpy, scipy, and other libraries via pip.
attractor_grid.py
Python 3 
# Attractor-based point grid with falloff
import rhinoscriptsyntax as rs
import Rhino.Geometry as rg
import math

# Inputs: pts (point list), attractor (point),
#         max_r (number), min_r (number)

results = []
radii   = []

for pt in pts:
    # distance from attractor
    dist = pt.DistanceTo(attractor)

    # normalize and invert (closer = larger)
    t = 1.0 - min(1.0, dist / 50.0)

    # smooth ease-in-out
    t_smooth = t * t * (3 - 2 * t)

    radius = min_r + (max_r - min_r) * t_smooth

    # create circle at point in XY plane
    plane  = rg.Plane(pt, rg.Vector3d.ZAxis)
    circle = rg.Circle(plane, radius)

    results.append(circle.ToNurbsCurve())
    radii.append(radius)

# Outputs: a (curves), b (radius values)
a = results
b = radii
